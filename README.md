📚 My Book Collection

A simple frontend project that showcases some of my favorite book recommendations.  
Built with **HTML** and **CSS**, this project highlights different books with their covers and titles.  

Features
- Static website built using HTML and CSS  
- Showcases a curated list of books with cover images  
- Clean and simple design for easy navigation  

Tech Stack
- HTML  
- CSS

- *Be Your Own Sunshine*  
- *Subconscious Mind*
